## Usage Notes
1. Open `index.html` in a browser. No build tools required.
2. Toggle the `<body>` class between `bg-warm`, `bg-balanced`, and `bg-cool` to preview Layer-0 backgrounds.
3. Switch card classes (`open`, `curtain`, `frame`, `seal`) to see gate behavior.
4. Explore `account.html` (token stub) and `library.html` (member grid with Curtain/Frame/Seal).
